Fliplet.UI.Table - Partial Selection UI

This demo showcases Fliplet.UI.Table's partial selection UI with indeterminate checkbox states using FontAwesome icons.

Partial Selection States

The select-all checkbox in the header shows three different states:

  • Empty (fa-square-o): No rows selected
  • Partial (fa-minus-square): Some rows selected
  • Full (fa-check-square): All visible rows selected
