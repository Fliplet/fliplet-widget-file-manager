Fliplet.UI.Table - Nested Tables Demo

This demo showcases how to use Fliplet.UI.Table within expanded rows to create nested table structures with async data loading.

Demo Features

  • Main Table: Shows departments with employee counts and row selection
  • Expanded Rows: Click on a row to expand and see team members
  • Async Loading: Employee data is loaded asynchronously (simulated 1-2 second delay)
  • Nested Table Features: Selection, sorting, and pagination in nested tables
  • Independent State: Each table (main and nested) maintains its own selection and sort state
  • Multi-level Selection: Select departments and/or individual employees
