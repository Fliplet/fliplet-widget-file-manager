Selection State Initialization

This demo shows three ways to initialize selection states:

  • Data-driven: Alice is pre-selected via _selected: true in row data
  • Data-driven: Carol is pre-marked partial via _partiallySelected: true
  • Configuration: Eve is pre-selected via initialSelection: [{ id: 5 }]
  • Configuration: David is pre-marked partial via initialPartialSelection: [{ id: 4 }]
