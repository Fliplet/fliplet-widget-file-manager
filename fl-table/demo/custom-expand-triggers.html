Fliplet.UI.Table - Custom Expand Triggers

This demo showcases different ways to implement expandable rows with custom UI elements as triggers, rather than dedicated expand columns.

1. Inline Chevron Triggers

Each name cell contains a chevron that users can click to expand the row. The chevron rotates when expanded.
